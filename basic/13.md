## while 文

while 文はインデントされているところの処理を、条件式が True の間処理を実行する構文です。

```py
# 無限に ぴえん と唱え続ける
while True:
    print('ぴえん')
```

```py
# 一度も実行しない
spam = False
while spam:
    print('spam')
```

```py
# 100 以上を数えるまで実行する
count = 0
while count < 100:
    count = count + 1
    print(count)
```

## for 文

for 文を用いると、リスト等から 1 つずつデータを取り出して、インデントされているところの処理を実行することができます。

```py
spam = [1,2,3,4,5]

for ham in spam:
    print(ham)
```
